## Getting Started


1. Create a project at [https://console.cloud.google.com/](https://console.cloud.google.com/).
2. Enable Google Maps
3. Paste the API key into `strings.xml`
   ```xml
   <string name="google_maps_key">GOOGLE_MAPS_KEY</string>
   ```
